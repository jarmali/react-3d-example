# 3D graphics example for React using babylon.js

##Installation and start
1. Clone repository
2. 'yarn'
3. 'yarn start'
4. Open browser at 'localhost:3000'

##Usage
Once 3D objects appear in browser, you can use mouse for rotation and wheel for zoom.
